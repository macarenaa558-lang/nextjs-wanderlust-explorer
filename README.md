# Referencias de Diseño

Antes de comenzar el desarrollo del proyecto, analicé distintas interfaces reales para definir el estilo visual y la experiencia de usuario que quería lograr. Estas aplicaciones sirvieron como inspiración para el diseño de la interfaz, la organización del contenido y la navegación.

## Airbnb

https://www.airbnb.com/

**Tomé como referencia:**

* Diseño limpio y moderno.
* Tarjetas con imágenes grandes e información clara.
* Barra de búsqueda y sistema de filtros.
* Navegación intuitiva y diseño responsivo.

## GetYourGuide

https://www.getyourguide.com/

**Tomé como referencia:**

* Organización del catálogo de experiencias.
* Distribución de la información dentro de las tarjetas.
* Sistema de categorías y filtros.
* Presentación clara de precios y valoraciones.

## Tripadvisor Experiences

https://www.tripadvisor.com/Attractions

**Tomé como referencia:**

* Listado de experiencias y destinos.
* Visualización de calificaciones y opiniones.
* Organización del contenido para facilitar la exploración.
* Diseño de tarjetas sencillo y fácil de recorrer.
